## 1. 詳細 UC
### UC6: エクスポート（concept §1.1 #6）
- トリガー: 「書き出し」→ 形式選択（画像 / Markdown / アウトライン）
- 処理: 現在のマップ（confirmed ノード中心、suggested は任意含む）を変換
- 出力: 画像 = キャンバス PNG/SVG、Markdown = 入れ子リスト、アウトライン = インデントテキスト

## 2. 入出力
| 形式 | 入力 | 出力 |
|---|---|---|
| 画像 | MapGraph | PNG/SVG（クライアント側 canvas/React Flow toImage） |
| Markdown | MapGraph | `.md`（ツリー入れ子 + kind 注記） |
| アウトライン | MapGraph | インデントテキスト |

### 副作用
- なし（読み取り + クライアント生成、ダウンロード）。データ可搬性（O12、§9.3）

## 3. データモデル
読み取りのみ（map-management の graph）

## 4. バリデーション + エラー
| 条件 | 振る舞い |
|---|---|
| CSV/区切り出力時 | injection エスケープ（`= + - @` 始まり、SEC-002） |
| Markdown 生成 | ユーザーテキストを安全にエスケープ（注入防止） |
| 空マップ | 空出力 + 案内 |

## 5. NFR + 連携
- クライアント完結（サーバ負荷なし）、大規模マップでもブラウザで生成
- 連携: map-management（graph）/ mindmap-canvas（画像化）/ _shared/ui
